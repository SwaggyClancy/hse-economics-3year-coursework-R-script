Сгенерировано: 2026-05-22 09:43:11

=== ТАБЛИЦЫ (output/tables/) ===

01_chain_stickiness.csv / .html
  Сводная таблица метрик ценовой липкости по каждой сети (Пятёрочка vs Магнит).
  Показатели: частота изменений, средний размер изменения, доля акций,
  глубина акции, средняя и медианная длительность ценового спелла, волатильность.

02_category_stickiness.csv / .html
  То же самое, но с разбивкой по категориям товаров внутри каждой сети.

03_cross_chain_correlations.csv / .html
  Корреляция недельной частоты изменений цен между Пятёрочкой и Магнитом
  по каждой категории. Высокая корреляция = синхронное поведение сетей.

04_intercategory_cors.csv
  Попарные корреляции между категориями внутри каждой сети.

05_intercategory_summary.csv
  Средняя и медианная межкатегорийная корреляция по каждой сети.

06_variance_decomposition.csv / .html
  Декомпозиция дисперсии изменений цен: сколько % объясняется сетью,
  категорией и магазином (последовательные R^2 adj. при добавлении FE).

07_cluster_centroids.csv / .html
  Центроиды K-means кластеров категорий по метрикам ценовой динамики.
  Позволяет выделить группы категорий со схожим поведением цен.

08_category_cluster_assignment.csv
  Принадлежность каждой категории к кластеру.

09_regression_results.tex
  Таблица панельных регрессий в формате LaTeX (для вставки в курсовую).
  Зависимая переменная: log-изменение эффективной цены.
  Регрессоры: принадлежность к Магниту, наличие промо-акции.

09_regression_coefs.csv
  То же самое в формате CSV.

10_regression_gof.csv / .html
  Метрики качества подгонки трёх спецификаций регрессии (N, R^2, R^2_adj).

11_robustness_coefs.csv
  Коэффициенты регрессии при проверках устойчивости (3 альтернативные спецификации):
  RC1 — только регулярные цены, RC2 — порог 0.5%, RC3 — без выбросов >50%.

11_robustness_table.csv / .html
  Итоговая сводная таблица robustness checks.
